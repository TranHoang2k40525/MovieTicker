# Cinema Bucket

Chức năng này xử lý rạp phim và các trang liên quan đến rạp.

## Vai Trò

- `domain`: cinema entity, repository contract, use case rạp gần đây và chi tiết rạp.
- `data`: model rạp và datasource lấy danh sách rạp.
- `presentation`: trang rạp gần đây, chi tiết rạp, và booking theo rạp.
