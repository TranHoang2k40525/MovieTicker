# Product Bucket

Chức năng này chứa combo và sản phẩm đi kèm.

## Vai Trò

- `domain`: product entity, repository contract, use case lấy danh sách sản phẩm.
- `data`: model sản phẩm và datasource lấy sản phẩm.
- `presentation`: màn hình catalog sản phẩm và chọn combo.
